الفصل الأول

مفاهيم الاحتمال الأساسية

(شرائح 4-15)

ما هو الاحتمال؟

الاحتمال في الحياة اليومية

الاحتمال هو مقياس لدرجة التأكد من وقوع حدث معين

التطبيقات في علوم الحاسوب:

  • تحليل الخوارزميات
  • أمان الشبكات
  • الذكاء الاصطناعي

فضاء العينة

التعريف: مجموعة جميع المخرجات المحتملة

أمثلة:

  • رمي النرد: Ω = {1, 2, 3, 4, 5, 6}
  • رمي العملة: Ω = {H, T}
  • حالة الخادم: Ω = {working, down, maintenance}

الأحداث

التعريف: مجموعة جزئية من فضاء العينة

مثال: رمي النرد

  • Event A = {2, 4, 6} (عدد زوجي)
  • Event B = {5, 6} (عدد > 4)
  • Event C = {1, 3, 5} (عدد فردي)

دالة كثافة الاحتمال

الشروط:

1. For each ω ∈ Ω: 0 ≤ p(ω) ≤ 1

2. Σ p(ω) = 1

مثال: رمي عملة منتظمة

p(H) = p(T) = 0.5

حساب احتمال الأحداث

القاعدة:

P(E) = Σ p(ω) for all ω ∈ E

مثال:

P(عدد زوجي) = p(2) + p(4) + p(6) = 1/6 + 1/6 + 1/6 = 1/2

قوانين الاحتمالات الأساسية

1. P(E ∪ F) = P(E) + P(F) - P(E ∩ F)

2. P(E̅) = 1 - P(E)

3. If E ⊆ F then P(E) ≤ P(F)

مثال - قانون الاتحاد

رمي النرد:

  • A = {2, 4, 6} (زوجي), P(A) = 1/2
  • B = {4, 5, 6} (≥ 4), P(B) = 1/2
  • A ∩ B = {4, 6}, P(A ∩ B) = 1/3
P(A ∪ B) = 1/2 + 1/2 - 1/3 = 2/3

التفسير التكراري

المفهوم: كلما زاد عدد التجارب، اقتربت النسبة النسبية من الاحتمال النظري

رمي العملة 1000 مرة:

Expected heads ≈ 500

Actual: 487, 503, 496, 512, etc.

اختيار فضاء العينة المناسب

مثال: رمي نردين

طريقة 1: Ω₁ = {2, 3, ..., 12} (المجموع)

طريقة 2: Ω₂ = {(1,1), (1,2), ..., (6,6)} (أزواج)

الطريقة 2 أفضل للحسابات!

الاحتمال المنتظم

عندما تكون جميع المخرجات متساوية:
p(ω) = 1/|Ω|

P(E) = |E|/|Ω|

أمثلة على الاحتمال المنتظم

  • سحب ورقة: P(Ace) = 4/52 = 1/13
  • رمي 3 عملات: P(exactly 2 heads) = 3/8
  • كلمة مرور 4 أرقام: P(all different) = 5040/10000

نظرية المجموعات

مبدأ الشمول والاستبعاد:

P(A ∪ B ∪ C) = P(A) + P(B) + P(C)
- P(A ∩ B) - P(A ∩ C) - P(B ∩ C)
+ P(A ∩ B ∩ C)

الفصل الثاني

فضاءات العينة المضروبة

(شرائح 16-25)

مبدأ الضرب

عند دمج تجربتين منفصلتين:
P(ω₁, ω₂) = p₁(ω₁) × p₂(ω₂)

مثال - رمي عملة ونرد

  • Ω₁ = {H, T}, Ω₂ = {1, 2, 3, 4, 5, 6}
  • Ω = Ω₁ × Ω₂ = {(H,1), (H,2), ..., (T,6)}
P(H, 3) = 1/2 × 1/6 = 1/12

فضاءات متعددة الأبعاد

Ω = Ω₁ × Ω₂ × ... × Ωₙ

p(ω₁, ω₂, ..., ωₙ) = p₁(ω₁) × p₂(ω₂) × ... × pₙ(ωₙ)

مثال - رمي 3 عملات

فضاء العينة:

Ω = {HHH, HHT, HTH, HTT, THH, THT, TTH, TTT}

P(HHT) = 1/2 × 1/2 × 1/2 = 1/8

P(exactly 2 heads) = 3/8

عمليات برنولي

تسلسل تجارب مستقلة ذات مخرجتين (نجاح/فشل)
  • احتمال النجاح = p
  • احتمال الفشل = q = 1-p
  • التجارب مستقلة

التوزيع الثنائي

b(k; n, p) = C(n,k) × p^k × (1-p)^(n-k)

مثال: رمي عملة 5 مرات

P(exactly 3 heads) = C(5,3) × (0.5)³ × (0.5)²
= 10 × 0.125 × 0.25 = 0.3125

أحداث الشكل المضروب

P(E₁ × E₂ × ... × Eₙ) = P(E₁) × P(E₂) × ... × P(Eₙ)

مثال - أحداث مضروبة

  • E₁ = {4, 5, 6} (نرد ≥ 4)
  • E₂ = {hearts} (ورقة قلوب)
P(E₁ × E₂) = 3/6 × 13/52 = 1/8

طرق عرض الأحداث

الطريقة الأولى: الحدث في الفضاء المركب

الطريقة الثانية: الحدث المقابل

Event in Ω₁: E₁ = {H}

Corresponding event: E₁* = {H} × Ω₂

P(E₁*) = P(E₁) = 1/2

تطبيق - شبكات الحاسوب

3 جدران حماية مستقلة، كل واحد يوقف 90% من الهجمات:

P(attack blocked) = 1 - P(all fail)
= 1 - (0.1)³ = 0.999

الفصل الثالث

الأحداث المستقلة والاحتمال الشرطي

(شرائح 26-40)

تعريف الاستقلالية

A and B independent ⟺ P(A ∩ B) = P(A) × P(B)

مثال - أحداث مستقلة

رمي عملة ونرد:

  • A = {H} (صورة), P(A) = 1/2
  • B = {6} (ستة), P(B) = 1/6
  • P(A ∩ B) = P(H, 6) = 1/12
  • P(A) × P(B) = 1/2 × 1/6 = 1/12 ✓

الاستقلالية ≠ الانفصال

الأحداث المنفصلة ذات الاحتمالات الموجبة ليست مستقلة:

A ∩ B = ∅ ⟹ P(A ∩ B) = 0

لكن P(A) × P(B) > 0

الاحتمال الشرطي

P(B|A) = P(B ∩ A) / P(A) where P(A) > 0

مثال - الاحتمال الشرطي

رمي النرد:

  • A = {2, 4, 6} (زوجي)
  • B = {4, 5, 6} (≥ 4)
P(B|A) = P({4, 6}) / P(A) = (2/6) / (3/6) = 2/3

الاحتمال الشرطي والاستقلالية

إذا كان A و B مستقلين:

P(B|A) = P(B)

نظرية الاحتمال الكلي

P(A) = Σᵢ P(A|Eᵢ) × P(Eᵢ)

حيث E₁, E₂, ..., Eₙ تشكل تقسيماً لـ Ω

مثال - نظرية الاحتمال الكلي

مصنع بثلاث خطوط إنتاج:

  • Line 1: 40% من الإنتاج، 2% معيب
  • Line 2: 35% من الإنتاج، 3% معيب
  • Line 3: 25% من الإنتاج، 1% معيب
P(معيب) = 0.02×0.40 + 0.03×0.35 + 0.01×0.25 = 0.021

قاعدة بايز

P(A|B) = P(B|A) × P(A) / P(B)

مثال - قاعدة بايز (التشخيص الطبي)

معطى:

  • P(مرض) = 0.01
  • P(إيجابي|مرض) = 0.95
  • P(إيجابي|سليم) = 0.05

مطلوب: P(مرض|إيجابي)

الحل: P(مرض|إيجابي) = 0.161

مثال - اختبار الجودة

  • Machine A: 60% من الإنتاج، 2% معيب
  • Machine B: 40% من الإنتاج، 5% معيب
  • منتج معيب تم اكتشافه
P(من Machine A | معيب) = 0.375

تطبيق - أمان المعلومات

  • مستخدمون عاديون: 95%، 1% فشل في المصادقة
  • مهاجمون: 5%، 80% فشل في المصادقة
  • فشلت عملية مصادقة
P(مهاجم | فشل) = 0.808

مشكلة الأبواب الثلاثة

  • 3 أبواب: خلف أحدها سيارة
  • تختار باب
  • المقدم يفتح باب آخر فارغ
  • هل تغير اختيارك؟
الجواب: نعم! P(win by switching) = 2/3

تطبيق - كشف الشذوذ

  • حركة مرور عادية: 99%، 0.1% مشبوهة
  • حركة هجوم: 1%، 90% مشبوهة
  • حزمة مشبوهة تم رصدها
P(هجوم | مشبوهة) = 0.901

تركيب الأحداث المشروطة

P(A ∩ B) = P(A) × P(B|A) = P(B) × P(A|B)

مثال - سحب بدون إرجاع:

P(both red) = P(R₁) × P(R₂|R₁) = 5/8 × 4/7 = 5/14
