🎮 المثال الثالث: الرسوم الحاسوبية والألعاب

1المشكلة

شخصية اللعبة في النقطة (5, 3) تحتاج إلى:

  • دوران 90° حول المركز
  • تكبير بمعامل 1.5
  • انتقال إلى (10, 15)
النقطة الأصلية: (5, 3)

2مصفوفات التحويل

🔄 مصفوفة الدوران 90°

R = [0 -1 0]
    [1 0 0]
    [0 0 1]

📏 مصفوفة التكبير 1.5

S = [1.5 0 0]
    [0 1.5 0]
    [0 0 1]

➡️ مصفوفة الانتقال

T = [1 0 10]
    [0 1 15]
    [0 0 1]

3التطبيق خطوة بخطوة

🎯 تطبيق التحويلات

الخطوة 1 - بعد الدوران: (-3, 5)
الخطوة 2 - بعد التكبير: (-4.5, 7.5)
الخطوة 3 - بعد الانتقال: (5.5, 22.5)

4النتيجة النهائية

🎯 الموقع النهائي للشخصية

(5.5, 22.5)

تم نقل الشخصية من (5, 3) إلى (5.5, 22.5) بنجاح!

نصيحة مهمة: ترتيب التحويلات مهم جداً! دائماً اقرأ من اليمين إلى اليسار: T × S × R × النقطة
🌐 المثال الرابع: شبكات التواصل الاجتماعي

1الشبكة الاجتماعية

أحمد
→
سارة
→
محمد
أحمد

محمد
العلاقات: أحمد يتابع سارة ومحمد، سارة تتابع محمد فقط

2مصفوفة التلاصق

        أحمد  سارة  محمد
أحمد  [0     1     1]
سارة  [0     0     1]
محمد  [0     0     0]
التفسير: 1 = يتابع، 0 = لا يتابع

3تحليل المسارات (A²)

🔍 المسارات غير المباشرة

A² = [0  0  1]
     [0  0  0]
     [0  0  0]
A²[1,3] = 1 يعني: أحمد → سارة → محمد

4التطبيقات العملية

🎯 اقتراح الأصدقاء

البحث عن A²[i,j] > 0 حيث A[i,j] = 0

📈 انتشار المحتوى

إذا نشر محمد → سارة ترى → أحمد يرى (67% وصول)

🌟 تحديد المؤثرين

محمد: مؤثر (in-degree = 2)
سارة: وسيط مهم

استخدامات حقيقية: فيسبوك، تويتر، لينكدإن يستخدمون نفس المبادئ لتحليل الشبكات وتوصيات المحتوى
