الاستقراء: الفكرة والمبدأ

  • أساس (Base Case) + خطوة انتقال (Inductive Step)
  • إذا صحت عند n وتنتقل إلى n+1 ⇒ تصح لكل n
  • تشبيه الدومينو (تحفيز سريع)

قالب البرهان بالاستقراء (Template)

  • حدِّد الخاصية P(n)
  • أثبت P(n₀) (الأساس)
  • افترض P(n) وأثبت P(n+1)
  • اختم بتطبيق المبدأ

الاستقراء — أمثلة خفيفة (1)

  • ∑i=0n i = n(n+1)/2
  • n+1 < n² لكل n>2 ⇒ 0 ≤ n²+n−1
  • n! > n² لكل n>4

الاستقراء — أمثلة خفيفة (2)

  • F₁+F₃+…+F_{2n−1} = F_{2n} − 1
  • |𝒫(X)| = 2^{|X|}
  • (1−x)(1+x+…+x^n) = 1−x^{n+1}

صحة البرامج: لماذا نثبت؟

  • الاختبار يكشف أخطاء لكنه لا يضمن الصحة العامة
  • الإثبات الرياضي يعطي ضمانًا شاملًا على كل المدخلات
  • نستخدم loop invariant + استقراء على عدد الدورات

Pseudocode مختصر + مثال خفيف

  • إسناد/شرط/تكرار/إخراج
  • مثال: تجميع عناصر مصفوفة — ثابت الحلقة: بعد k دورات: sum = Σ a[i] من 1 إلى k

√17 — طريقتان (مختصر)

  • طريقة I: بناء خانة بخانة — حد خطأ Eₙ < 10^{-n}
  • طريقة II (نيوتن): R_{n+1} = (R_n + 17/R_n)/2 — تقارب تربيعي
  • مثال عددي: R₀ = 4 ⇒ R₁ ≈ 4.125

العلاقات الثنائية: تعريف وتمثيل

  • علاقة على X: R ⊆ X×X (مجموعة أزواج مرتّبة)
  • تمثيل: أزواج / جدول أو مصفوفة / رسم موجّه (digraph)
  • مثال خفيف: ≤ على {1,2,3} كجدول أزواج

خصائص العلاقات (تصنيف سريع)

  • انعكاسية/لاإنعكاسية
  • تماثلية/لاإتماثلية/مضادّة للتماثل
  • انتقالية — تذكير: الفرق بين Asymmetric وAntisymmetric

تكافؤ وترتيبات + Hasse

  • Equivalence: Reflexive + Symmetric + Transitive ⇒ فئات تكافؤ
  • Partial/Total Order: PO ومتى يكون كليًا (قابلية المقارنة)
  • مثال: (𝒫({a,b}), ⊆) ورسم Hasse مبسّط

عمليات وإغلاقات + تمثيل مصفوفي

  • اتحاد/تقاطع/متمم، معكوس R^{-1}، تركيب R∘S، قوى R^n
  • إغلاقات: انعكاسي R∪I — تماثلي R∪R^{-1} — انتقالي R⁺
  • مصفوفة التلاصق، ضرب بولياني، Warshall (Reachability)

أمثلة خفيفة + توجيهات مذاكرة

  • أوراق اللعب: HigherValue (لاإنعكاسي/انتقالي)، SameSuit (تكافؤ)
  • تمارين:
    • صنّف a≡b (mod 3) (فئات البواقي)
    • هل ≤ تماثلية؟ أم مضادّة للتماثل؟
  • راجِع الأمثلة والقراءات المحددة

الخاتمة

  • الخلاصة: الاستقراء (أداة البرهان) • صحة البرامج (ثابت الحلقة) • العلاقات (تعريف/تصنيف/تمثيل)
